✅ ШАГ 2: Vercel Deploy (ОСТАЛОСЬ 3 КЛИКА!)

   2.1. Перейти на https://vercel.com

   2.2. Нажать: New Project

   2.3. Выбрать GitHub → Авторизоваться

   2.4. Выбрать "sky-project-2024" репо

   2.5. Нажать "Import"

   2.6. ВАЖНО! Добавить Environment Variables:
        PORT = 3000
        NODE_ENV = production
        ALLOWED_ORIGINS = https://your-vercel-app.vercel.app
        TELEGRAM_BOT_TOKEN = your_token_here
        TELEGRAM_CHAT_ID = your_chat_id_here

   2.7. Нажать "Deploy"

   ⏳ Подождать 2-3 минуты...

✅ ШАГ 3: Verify (ПРОВЕРИТЬ)

   - Homepage: https://your-project.vercel.app
   - API health: https://your-project.vercel.app/api/health
   - Contact form → должна отправить в Telegram
